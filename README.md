# system-audio-share-onwindows
A automatic way of sharing your system audio altogether with your system audio

You must follow all the steps bellow to make it work:
1. Install voicemeeter banana: https://vb-audio.com/Voicemeeter/banana.htm
2. Install voicemeeter vb-cable drivers: https://vb-audio.com/Cable/VirtualCables.htm
3. change your main output audio device in the system to "CABLE Input (VB-Audio Virtual Cable)
4. change your main microphone device in the system to "Voicemeeter Out B1"
5. Download the .exe file in the releases page and execute it, it sould work.
